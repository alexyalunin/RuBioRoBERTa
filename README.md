
# RuBioRoBERTa

1. Download RuMedBench and install `requirements.txt`

2. For pre-training run `scripts/1_pretrain.sh` or `scripts/1_pretrain_distributed.sh`

3. For fine-tuning on RuMedBench run `scripts/2_RuMedBench.sh`
